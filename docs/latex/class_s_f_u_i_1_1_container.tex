Draw the component or inner components on an overlay layer on top of the main UI tree to the render target. 

This is relevant for components that are actively animating and do not want their drawn geometry subject to clipping by their parents\textquotesingle{} bounds. It is also useful for inner components like tooltips, context menus, modals, and other special UI components.
